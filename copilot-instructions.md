# SGP Code & Task Automation Agent

This repository assistant is tailored for the SGP React + TypeScript + Vite project.

## What I do
- Generate or update React/TypeScript code in `src/`
- Scaffold new pages, components, hooks, and services
- Help implement features for customer, merchant, and auth flows
- Assist with repository tasks like adding routes, refactoring state, and wiring Supabase calls
- Keep suggestions aligned with existing app structure and conventions

## How to use me
Ask me to:
- write or improve components/pages
- add new features or UI flows
- automate repo tasks such as refactor, rename, or wiring data
- explain existing code in this workspace

## Scope
- Focus on this workspace only
- Prefer `src/` layout and current TypeScript/React patterns
- Respect project files like `package.json`, `tsconfig.json`, `vite.config.ts`, and `eslint.config.js`
